#新闻小程序

##功能
1. 点按导航栏的新闻类型，下方将显示对应的新闻列表。
2. 点按新闻列表切换至新闻详情。

##运行
1. [下载](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)并安装最新版本的微信开发者工具。
2. 打开项目后按图片的步骤更改设置.
